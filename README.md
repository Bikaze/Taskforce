## 🏗 Setup & Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/majestic-wallet.git
```

2.Install dependencies

```bash
cd majestic-wallet
npm install
```

3.Start the development server

```bash
npm run dev
```
